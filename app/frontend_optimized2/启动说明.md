# 职业规划智能体前端 - 快速启动脚本

## Windows 启动方式

### 方式一：命令行启动

```bash
# 1. 进入项目目录
cd app/frontend_optimized2

# 2. 安装依赖（首次运行）
npm install

# 3. 启动开发服务器
npm run dev
```

### 方式二：使用批处理文件

双击运行 `start.bat` 文件即可自动启动。

## 访问地址

启动后访问：http://localhost:3000

## 注意事项

1. 确保后端服务已在 `http://localhost:8000` 运行
2. 首次使用需要上传简历以创建会话
3. 岗位分析可能需要 30-60 秒等待时间
